## About Code in Place
Code in Place is a free introductory CS program by Stanford University that teaches programming fundamentals using Python and Karel the Robot.

## What this repository contains
- Karel problem solutions
- Practice exercises from lectures and assignments
- Programs using loops, conditionals, and decomposition
- Solutions designed to work across multiple world configurations

## Skills demonstrated
- Problem decomposition
- Control flow (while loops, for loops)
- Debugging under constraints
- Writing clean and readable code
